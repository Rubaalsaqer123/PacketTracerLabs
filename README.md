# PacketTracer Labs Repository

Created by Ruba Alqahtani

This repository contains a collection of Cisco Packet Tracer lab files designed to help students and beginners practice essential networking concepts, configurations, and troubleshooting skills.

---

## Project Overview

The goal of this project is to provide practical networking exercises that support learning topics commonly covered in CCNA and other network-related courses.  
Each lab focuses on a specific concept and includes the necessary Packet Tracer (.pkt) file.

---

## Lab Categories

- Basic Networking (IP addressing, subnetting, connectivity tests)  
- Switching (VLANs, trunking, spanning-tree configuration)  
- Routing
- Network Security (ACLs, port security)  
- WAN Technologies (PPP, Frame Relay, and others)

---

## How to Use

1. Download any .pkt file from the repository.  
2. Open it using Cisco Packet Tracer.  
3. Follow the instructions included in the lab file.  
4. Configure the network and verify your work.

---

## Requirements

- Cisco Packet Tracer (latest version recommended)
